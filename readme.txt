== Description ==

You need to be a registered member of [SMSGatewayCenter.com](https://www.smsgateway.center.com) to use this plugin. Install this simple plugin and start sending bulk SMS. This gateway supports India and International both.

SGCSMS allows you to send bulk SMS from your Wordpress site.


== Feature ==

* **Send single/bacth SMS**: You can include comma separated mobile numbers and can send upto 2000 multiple recipients at once amd Send Unicode and Text SMS message from Wordpress sites.
* **Send Group SMS**: Send SMS to your saved group on SMS Gateway Center. You can send to multiple groups at once.
* **Send SMS to Subscribers**: Send SMS with any content to your locally saved groups and contacts.
* **Check account balance and validity**: Check SMS Balance and Account validity.
* **View profile**: View your saved profile of SMS Gateway Center.
* **Message Templates**: View all your added DLT templates.
* **Groups**: View all group list with total contacts saved from our portal .
* **Change password**: Change password from your wordpress site.
* **incoming SMS report**: You can view your incoming SMS report in this section.
* **SMS delivery report**: You can view your sent SMS report in this section.
* **Credit summary report**: You can view all your purchases and adjustments in this report.
* **Subscriber Group**:  Create unlimited group within your wordpress based site.
* **Subscriber Contacts**:  Create unlimited contacts and assigned to group within your wordpress based site.
* **Sender Name**: View all your sender names.
* **Settings**: Authenticate your username and password in settings. Add default sender name and routing.

== Installation ==

= AUTOMATIC INSTALLATION =

1. Log in to your WordPress blog and visit Plugins->Add New.
2. Search for SGCSMS Bulk SMS Sender, click "Install Now" and then Activate the Plugin
3. Visit the "SGCSMS -> Settings" menu.
4. Configure the options with your registered username and password
5. Visit the "SGCSMS -> Send Bulk SMS" menu to send bulk SMS.
6. Manually send bulk SMS.


= MANUAL INSTALLATION =

1. Copy the entire /sms-gateway-center-bulk-sms-sender/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. Visit the "SGCSMS -> Settings" menu.
4. Configure the options with your registered username and password
5. Visit the "SGCSMS -> Send Bulk SMS" menu to send bulk SMS.
6. Manually send bulk SMS.

== Frequently Asked Questions ==

= Do you provide free demo with some test credits? =

Yes, we do. Go to [sign up](https://www.smsgateway.center) and select Instant demo.


= Where can I find my Username and Password? =

You need to be a registered member with [SMS Gateway Center](https://www.smsgateway.center) and your account status needs to be active in order to work with this plugin. You can sign up on our portal and use it here.

= Does SMS Gateway Center provide International and Indian SMS Gateways? =

Yes, we offer all countries SMS delivery. You can contact us for pricing.

= How should I use mobile numbers to mobile number column? =

You need to use 10, 11 or 12 digits Indian mobile number as 09999999999 or 9999999999 or 919999999999. If you are using international route, then prefixing country code is mandatory.

= Can I send to multiple mobile numbers? =

Yes, you can, just comma separate the mobile numbers as 9999999991,919999999992, 09999999993

= Is there any limit while sending bulk SMS? =

Actually no, but we recommend you to use maximum 1000 mobile numbers in mobile number field.

= Is there any limit on Incoming SMS report? =

No, We support 30 days retrieval and you can see all those 30 days report. Rest of the reports will be archived and stored in our server for a period of 1 year.

= Does any of the information gets inserted in the database? =

No, We do not record any data in your wordpress database, except the settings related fields.

= What should I use in settings page? =

Just use your registered username and password in the settings page and select default settings once your login is authenticated.

= Why Sender names are not listed in the drop-down? =

If sender names are not appeared in the drop-down then no sender names have been approved in your account. Contact us for more details.

= How can I send DLT Template ID? =

Our system is capable enough to scan and assign a DLT template ID before sending out to the operators even if you are not adding the DLT template ID while sending. This option only will be shown to Indian clients.

= How can I share files over SMS such as Images/Docs/Videos?  =

You can follow our tutorial [here](https://www.smsgatewaycenter.com/sms-file-attachments.html)

= Do you provide Google Verified SMS?  =

Yes, we do. You can request to enable the same with our team. [More info here](https://www.smsgatewaycenter.com/verified-sms.html)
